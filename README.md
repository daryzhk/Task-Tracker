<h1 align="center">Task-Tracker</h1>

<p align="center">
  Консольное приложение-ежедневник для задач.
</p>

## Описание
Task Tracker - это консольное приложение-ежедневник, позволяющее создавать, редактировать и отслеживать задачи.
<p>Проект демонстрирует базовые принципы работы с базами данных, обработку пользовательского ввода и реализацию CRUD-логики.</p>

## Основные функции
- Добавление задач
- Просмотр списка задач
- Редактирование задач
- Удаление задач
- Работа с дедлайнами:
  - проверка формата даты
  - запрет установки даты в прошлом
- Cтатистика:
  - общее количество задач
  - количество активных и завершённых
  - задачи с просроченным дедлайном
  - процент выполнения
 
## Используемые технологии
- Python
- Библиотеки SQLite и datetime.

## Особенности реализации
- Реализация полного CRUD-цикла (создание, просмотр, редактирование, удаление)
- Работа с SQLite и параметризованными SQL-запросами
- Валидация пользовательского ввода
- Работа с датами и дедлайнами (datetime)
- Проверка просроченных задач
- Реализация статистики и аналитики (включая процент выполнения)
- Обработка ошибок без прерывания работы программы
- Организация консольного интерфейса и логики взаимодействия с пользователем

## Установка и запуск
1. Для работы нужен Python v3
2. Скачайте файл task_tracker.py или склонируйте репозиторий:
```
git clone https://github.com/daryzhk/task-tracker.git
cd task-tracker
```
3. Запустите приложение:
```python task_tracker.py```
